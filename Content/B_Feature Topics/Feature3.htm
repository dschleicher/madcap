Cloudian users are advised to use the HTML version of the Cloudian documentation set, which is included in the product package. There is a great deal of useful hyperlinking between the various Cloudian documents. These cross-document links work in the HTML documents but not in the PDF documents.

However, you may wish to use PDF versions Cloudian documents for printing purposes. These PDF files are available on a request basis.

Cloudian documentation PDFs are optimized for the paper size "A4", the most common paper size internationally.

If you are printing to Letter size paper (8.5" x 11"), in your Adobe Reader or Adobe Acrobat's Print dialog use the "Fit to Page" option. Otherwise page headers may get cut off in the print-outs.
